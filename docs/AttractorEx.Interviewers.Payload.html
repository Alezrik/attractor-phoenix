    AttractorEx.Interviewers.Payload (attractor_phoenix v0.1.0)

    Copy Markdown View Source

    Shared question and answer normalization for interviewer adapters.

    This module keeps console, queue, callback, recording, and HTTP interviewers on the same wire format so wait.human behavior stays consistent across transports.
