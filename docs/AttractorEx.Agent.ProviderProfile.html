    AttractorEx.Agent.ProviderProfile (attractor_phoenix v0.1.0)

    Copy Markdown View Source

    Provider-aligned configuration for the coding-agent loop.

    A profile packages a model, toolset, provider options, and an optional system-prompt builder so agent sessions can stay portable across providers.

    The module also exposes a maintained cross-provider integration matrix for the built-in OpenAI, Anthropic, and Gemini presets.
