# `AttractorPhoenix`
[🔗](https://github.com/Alezrik/attractor-phoenix/blob/main/lib/attractor_phoenix.ex#L1)

AttractorPhoenix keeps the contexts that define your domain
and business logic.

Contexts are also responsible for managing your data, regardless
if it comes from the database, an external API or others.
