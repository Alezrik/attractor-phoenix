    AttractorEx.HTTP.Manager (attractor_phoenix v0.1.0)

    Copy Markdown View Source

    GenServer that owns the in-memory state for HTTP-managed pipeline runs.

    It tracks pipeline status, events, checkpoints, pending human questions, and event subscribers, and acts as the coordination point between the engine and the router.
